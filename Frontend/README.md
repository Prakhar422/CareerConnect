# 🎨 Career Connect - Frontend

Frontend of Career Connect built using React and Vite, providing a responsive and user-friendly interface.

---

## 🌐 Live Demo

* 🚀 https://career-connect-client-rho.vercel.app

---

## 🚀 Features

* 🔐 Clerk Authentication
* 🔍 Job search & filtering
* 📝 Quill editor integration
* 📅 Day.js for date formatting
* 🔔 Toast notifications
* 🎯 Responsive UI

---

## 🛠️ Tech Stack

* React.js (Vite)
* Tailwind CSS
* React Router DOM

---

## 📦 Key Dependencies

* Axios
* @clerk/react
* Quill
* Day.js
* React Toastify
* Lucide React
